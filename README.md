# Gamesphere-Tournament-Hub
A cloud-based esports tournament management platform where players can register teams, join tournaments, track match schedules, and view live leaderboards using ASP.NET and AWS services.

## Prerequisites

- Git
- .NET 10 SDK (for the ASP.NET backend in `backend/`)
- Node.js 22.12+ and npm (for the React frontend in `frontend/`)
- Visual Studio 2022 or VS Code (recommended)

Optional for cloud deployment:

- AWS CLI
- An AWS account with access to required services (for example RDS, EC2, S3)
